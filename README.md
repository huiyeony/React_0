# 3. Key Features (주요 기능)


- **CRUD 기능**:
  - create : 새로운 할일을 작성하고, 로컬 스토리지에 저장합니다.
  - read : 작성된 할일 목록을 확인 할 수 있습니다.
  - update : 할일을 수정하고 로컬 스토리지에 저장 할 수 있다.
  - delete : 할일을 삭제하고 로컬 스토리지에 저장 할 수 있다.
  

- **키워드 검색**:
  - 검색 키워드를 포함하는 아이템만 필터링 하여 리스트 형태로 보여줍니다.
